# 🏠 House Rent Prediction System

A Machine Learning web application that predicts house rent based on property features such as BHK, Size, Bathroom, City, Furnishing Status, and more.

##  Live Demo

https://house-rent-prediction-nishantsingh.streamlit.app/

##  Features

- Predict house rent instantly
- User-friendly Streamlit interface
- Random Forest Regressor model
- Real-time predictions
- Deployed on Streamlit Cloud

##  Dataset

- House Rent Dataset
- 4746+ records
- Features:
  - BHK
  - Size
  - Bathroom
  - Area Type
  - City
  - Furnishing Status
  - Tenant Preferred
  - Point of Contact

##  Technologies Used

- Python
- Pandas
- NumPy
- Scikit-Learn
- Streamlit
- Joblib

##  Data Preprocessing

- Missing value check
- Duplicate removal
- Outlier removal using IQR
- One-Hot Encoding
- Feature Selection

##  Models Tested

| Model | R² Score |
|---------|---------|
| Linear Regression | 0.68 |
| KNN Regressor | 0.38 |
| Random Forest Regressor | 0.84 |

## 🏆 Best Model

Random Forest Regressor achieved the highest performance with an R² score of approximately 0.84.
